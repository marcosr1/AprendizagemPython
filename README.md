# 📊 Modelagem Matemática com Python

Este repositório foi criado com o objetivo de servir como um **ambiente de estudo, laboratório e aprendizado prático** em modelagem matemática utilizando Python.

A proposta é explorar conceitos matemáticos aplicados ao mundo real, transformando equações em código e visualizações, permitindo uma compreensão mais profunda tanto da matemática quanto da programação.

---

## 🎯 Objetivos

* Praticar **modelagem matemática aplicada**
* Desenvolver habilidades em **Python para ciência de dados**
* Visualizar comportamentos matemáticos através de gráficos
* Construir uma base sólida para áreas como:

  * Inteligência Artificial
  * Análise de Dados
  * Simulações computacionais

---

## 🧪 O que já foi desenvolvido

Atualmente, o repositório contém os seguintes estudos e implementações:

### 📈 Crescimento Populacional (Modelo Exponencial)

* Implementação do modelo matemático de crescimento populacional
* Uso de funções exponenciais
* Geração de gráficos para visualização do crescimento ao longo do tempo

### 📉 Simulação de Queda Livre

* Aplicação de equações da física para modelar movimento
* Cálculo da posição ao longo do tempo
* Visualização gráfica do comportamento da queda
